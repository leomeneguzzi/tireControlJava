Name Description
bandtype This table store tires band types.
mount This table is used to set the times when the tire is mount. The relation with retread is because the mount is based in the retread, not in the tire.
mountlocation Physical location of the tire on the truck where it was mounted.
retread This table store the retreads made on the tire.
tire This table represent the tire's carcass.
tirebrand This table store tires brands.
tiremodel This table store tires models.
tiresituation This table is designed to set a current tire's situation. Ex: mounted, unmounted, died, unavailable, etc.
truck Used to identify the truck that the tire will be mounted.
unmount This table is used to store an unmount.
unmountreason The unmount have a reason. Ex: mechanical problems, end of life, factory problem, etc.
